# 题目名称：单词搜索2

## 题目链接：https://leetcode-cn.com/problems/word-search-ii/


## 题解：

给定一个二维网格和一个单词列表，查出所有在二维网格中出现的单词，这个“在”的定义为：单词按照字母顺序，并通过相邻单元格内的字母构成，同一个单元格在一个单词中不能重复使用。

先从一个简单的思路，一并给出题目的详细解释。有一个单词列表，就可以逐一对每一个单词在该网格中进行查找，查找方法为，从某一个点作为第一个字母元素，比对相同与否，然后向四周（上下左右）遍历，寻找与第二个相同的，然后不断往后，注意不能重复，当没有时即不存在。

这样一来，每一个单词都要对网格进行完整的查询，所以浪费了很多时间。

换一种思路，我们可以优先查找网格，然后与所有单词逐一比较，去除不能匹配的单词，当没有单词能匹配时则说明该路径不可以，需要回溯。

至于单词的匹配方法，可以使用前缀树的数据结构，相当于一个广度优先遍历该树，留下匹配的分支，然后往后。这样的好处是，对于前缀相同的，必然我们可以同时留下这些分支，而不用反复匹配。
